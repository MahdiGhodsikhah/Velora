**علت:** متغیر `$_SESSION['role']` تعریف نشده بود  
**راه‌حل:** به‌روزرسانی `AuthController.php` و `navbar.php`

**علت:** کاربر نقش admin نداشت  
**راه‌حل:** ابزار تبدیل کاربر به ادمین

### ✅ 3. ThemeController بارگذاری نمی‌شد
**علت:** فایل در تست بارگذاری نشده بود  
**راه‌حل:** به‌روزرسانی `test-theme.php`

### مرحله 2️⃣: تبدیل کاربر به ادمین

```

```

1. نام کاربری خود را وارد کنید (مثلاً `mahdi`)
2. یا از دکمه‌های انتخاب سریع استفاده کنید

---

### مرحله 3️⃣: خروج و ورود مجدد

## 🎓 نکات مهم

### 1. تفاوت `role` و `user_role`

در کد:
- `$_SESSION['role']` - استفاده در AdminController و navbar
- `$_SESSION['user_role']` - برای سازگاری با کد قبلی

هر دو مقدار یکسانی دارند.

### 2. نقش‌های موجود

```php
'admin'     -> دسترسی به پنل مدیریت
'customer'  -> کاربر عادی
'moderator' -> (آینده) مدیر محتوا
```

### 3. بررسی دسترسی

```php
if (isset($_SESSION['role']) && $_SESSION['role'] === 'admin') {
    // دسترسی به پنل ادمین
}
```
