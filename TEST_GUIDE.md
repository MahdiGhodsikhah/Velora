### 🐛 مشکلات رایج و راه‌حل

#### مشکل 1: خطای 500
```
علت: .htaccess یا PHP
راه‌حل:
1. چک کنید mod_rewrite فعال باشد
2. فایل .htaccess صحیح باشد
3. لاگ Apache را بررسی کنید
```

#### مشکل 2: اسلایدر کار نمی‌کند
```
علت: jQuery یا Slick لود نشده
راه‌حل:
1. Console مرورگر را چک کنید
2. مسیر فایل‌های JS را بررسی کنید
3. ترتیب لود اسکریپت‌ها (اول jQuery، بعد Slick، بعد main.js)
```

#### مشکل 3: تصاویر نمایش داده نمی‌شوند
```
علت: مسیر اشتباه
راه‌حل:
1. چک کنید BASE_URL صحیح باشد
2. تصاویر در پوشه public/assets/images قرار دارند
```

#### مشکل 4: فوتر انیمیت نمی‌شود
```
علت: IntersectionObserver
راه‌حل:
1. Console را چک کنید
2. اسکرول کنید تا فوتر به viewport برسد
3. اگر نشد، مرورگر را refresh کنید
```

#### مشکل 5: منوی dropdown باز نمی‌شود
```
علت: JavaScript یا CSS
راه‌حل:
1. Console خطا را چک کنید
2. مطمئن شوید main.js لود شده
3. CSS header-animated.css لود شده باشد
```
