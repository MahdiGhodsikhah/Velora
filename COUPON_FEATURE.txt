📱 نحوه کار:
-------------

کاربر در صفحه سبد خرید:
1. کد تخفیف را وارد می‌کند
2. روی دکمه "اعمال" کلیک می‌کند
3. سیستم بررسی می‌کند:
   
   ✅ اگر خالی باشد:
      → پیغام قرمز: "لطفا کد تخفیف را وارد کنید"
   
   ✅ اگر پر باشد:
      → ارسال به سرور
      → دریافت پاسخ
      
      اگر معتبر بود (success: true):
      → پیغام سبز: "کد تخفیف با موفقیت اعمال شد"
      → پاک شدن input
      
      اگر نامعتبر بود (success: false):
      → پیغام قرمز: "کد تخفیف نامعتبر است یا منقضی شده"

🔮 توسعه آینده:
-----------------

برای فعال‌سازی کامل سیستم کد تخفیف:

1. ایجاد جدول coupons:
   CREATE TABLE coupons (
     id INT PRIMARY KEY AUTO_INCREMENT,
     code VARCHAR(50) UNIQUE NOT NULL,
     discount_type ENUM('percent', 'fixed'),
     discount_value INT NOT NULL,
     min_purchase INT DEFAULT 0,
     max_uses INT DEFAULT NULL,
     used_count INT DEFAULT 0,
     valid_from DATE,
     valid_until DATE,
     is_active BOOLEAN DEFAULT 1
   );

2. پیاده‌سازی متد validateCoupon() در CartController

3. ذخیره کد تخفیف در session

4. اعمال تخفیف در محاسبات checkout

5. نمایش مقدار تخفیف در UI

📝 کد نمونه برای آینده:
--------------------------

// در CartController.php
private function validateCoupon($code) {
    $code = db_escape($code);
    $today = date('Y-m-d');
    
    $sql = "SELECT * FROM coupons 
            WHERE code = '$code' 
            AND is_active = 1 
            AND (valid_from IS NULL OR valid_from <= '$today')
            AND (valid_until IS NULL OR valid_until >= '$today')
            AND (max_uses IS NULL OR used_count < max_uses)
            LIMIT 1";
    
    return db_fetch_one($sql);
}
